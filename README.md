# Crypto Market Data Analysis

---

## Description 

Utilize unsupervised machine learning to analyze various cryptocurrency investment opportunities grouped into potential portfolios clustered based on K-Means clustering. 

---

## Requirements

Python 3.7
Pandas
hvPlot
sklearn
pathlib
